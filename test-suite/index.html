JSON-LD Test Suite

Overview

The JSON-LD Test Suite provides an interactive mechanism to check the conformance of an JSON-LD processor and SPARQL query engine. This service is intended for anybody that wants to see how JSON-LD works as well as developers that want to do conformance testing on their own JSON-LD/SPARQL implementations.

There are several things that you can do on this web page:

  • To run a unit test, click on the [TEST] link.
  • To see the details of a unit test, click on the [show details] link.
  • You can execute every unit test by clicking on the [Run All Unit Tests] button.
  • You can modify the Test Suite, Approval Status, JSON-LD Extractor Service and SPARQL Engine URLs.

Status

  • These tests are in active development by the JSON-LD Working Group at the World Wide Web Consortium.
  • These tests may be used to check conformance for an JSON-LD processor.
  • The tests are subject to change at any point between now and June 2011.
  • All bugs, suggestions and thoughts about The JSON-LD Test Suite should be sent to the JSON-LD Working Group mailing list.
